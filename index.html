This is a simple experiment which shows the current live subscriber count on YouTube for a user. The count displayed on YouTube is often incorrect because it doesn't update in realtime. The count on this page is taken directly from the API and hence is garaunteed to be accurate.

You can see for yourself how inaccurate the count on YouTube is, never trust it! You can use this tool to link to any YouTube channel this way: http://akshatmittal.github.io/youtube-realtime/#!/<username> or share the current URL for this YouTuber!

Btw, you can just bookmark the URL if you ever want to check the live count again!

The live count is updated every second, while count from youtube is updated every minute.
