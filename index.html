This is a simple experiment which shows the current live subscriber count on YouTube for a user. The count displayed on YouTube is often incorrect because it doesn't update in realtime. The count on this page is taken directly from the API and hence is guaranteed to be accurate.

After clicking change username above, you need to enter the username if you have a /user/ link, or enter the channel ID if you have a /channel/ type URL. Eg: If your channel link is http://www.youtube.com/user/SkyDoesMinecraft, you just need to enter SkyDoesMinecraft. If you channel link is http://www.youtube.com/channel/UCKlhpmbHGxBE6uw9B_uLeqQ, you just need to enter UCKlhpmbHGxBE6uw9B_uLeqQ when prompted after clicking the current username.

You can see for yourself how inaccurate the count on YouTube is, never trust it! You can use this tool to link to any YouTube channel using either one of the following URL schemes:
  • http://akshatmittal.github.io/youtube-realtime/#!/<username or channel ID>
  • youtuberealtime.ga/<username or channel ID>
Or you can just copy the current URL to share the count for this YouTuber! You can also bookmark this page if you ever want to check the live count again!

The live count is updated every second, while count from YouTube is updated every minute.
